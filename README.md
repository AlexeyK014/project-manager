# 📋 Project Manager

Современное веб-приложение для управления проектами и задачами.

Проект позволяет создавать проекты, организовывать работу с задачами, отслеживать статус выполнения и управлять рабочим процессом через удобный интерфейс.

---

## 🚀 Основные возможности

### Управление проектами

- Создание проектов
- Редактирование информации о проекте
- Удаление проектов
- Просмотр списка проектов

### Управление задачами

- Создание задач
- Редактирование задач
- Удаление задач
- Изменение статуса выполнения
- Назначение задач проектам

### Пользовательский интерфейс

- Современный адаптивный дизайн
- Работа на мобильных устройствах
- Интуитивная навигация
- Быстрая работа SPA без перезагрузки страниц
